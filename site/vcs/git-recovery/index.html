Undo Và Recovery Trong Git

Phần này gom các kỹ thuật hoàn tác, phục hồi và phần tự luyện cuối chương.

Undo & Recovery

Bỏ file khỏi staging

git restore --staged file.txt

Bỏ thay đổi

git restore file.txt

Quay lại commit trước

git reset HEAD~1

Reset mạnh

git reset --hard HEAD~1

Khôi phục bằng reflog

git reflog
git reset --hard HEAD@{2}

Mẹo quan trọng

git reflog lưu lại gần như mọi thao tác Git. Bạn có thể khôi phục commit kể cả khi đã reset --hard.
