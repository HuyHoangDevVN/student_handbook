Student IT Handbook

FIT DNU Handbook

Handbook định hướng thực tập và workflow làm việc thực tế

Student IT Handbook được viết để giúp sinh viên không bị ngợp khi bước vào repo, team và task thật. Nội dung tập trung vào cách tiếp cận dự án, setup môi trường, Git, database, API, Docker, CI/CD, troubleshooting và cheatsheet, xoay quanh case study InternHub API.

Start Here Quickstart

Đọc để định hướng, dùng để tra cứu, và quay lại khi gặp task hay lỗi thực tế.

Handbook này dùng khi nào

Chuẩn bị đi thực tập Đọc checklist nếu bạn muốn biết cần chuẩn bị gì trước khi vào team. Mới vào repo và nhận task Đọc GitHub Workflow nếu bạn sắp tạo branch, mở PR và nhận review. Gặp lỗi và cần tra cứu nhanh Mở Troubleshooting và Cheat Sheets khi cần command, checklist và guardrail.

Các nhóm nội dung chính

01

Định hướng

Start Here, Quickstart, Checklist sinh viên mới và Sample Project để biết handbook dùng để làm gì.

02

Làm việc trong team

Terminal, Linux, Git, GitHub Workflow và editor setup để không bị ngợp khi vào repo thật.

03

Làm việc với dự án

Database, HTTP API, API testing, Docker, Docker Compose và dependency management.

04

Làm việc hiệu quả hơn

CI/CD, deployment, monitoring, security, troubleshooting và cheatsheet để tiếp cận production mindset.

Điểm vào khuyến nghị

Start Here Dùng nếu bạn chưa biết nên đọc handbook theo tình huống nào. InternHub API Dùng nếu bạn muốn giữ bối cảnh chung cho các ví dụ kỹ thuật. Cheat Sheets Dùng khi bạn đã học nền và cần tra cứu command nhanh.
