Tìm Kiếm Và Biến Môi Trường

Phần này gom các lệnh tìm kiếm, biến môi trường, mẹo hữu ích và phần tự luyện.

Tìm kiếm file

Tìm file theo tên

find . -name "*.js"

Tìm text trong file

grep "TODO" -r src/

Tìm kèm số dòng

grep -rn "error" .

Tìm theo extension

grep -rn "error" --include="*.log" .

Ripgrep (nhanh hơn)

rg "TODO" src/

Biến môi trường

Xem biến

echo $HOME
echo $PATH

Tạo biến tạm

export MY_VAR="hello"

Sử dụng

echo $MY_VAR

Lưu vĩnh viễn

Thêm vào:

~/.bashrc

hoặc

~/.zshrc

Ví dụ:

echo 'export MY_VAR="hello"' >> ~/.bashrc

Sau đó reload:

source ~/.bashrc

Lệnh hữu ích khác

which python

Đường dẫn binary.

whoami

User hiện tại.

df -h

Dung lượng disk.

du -sh folder/

Kích thước folder.

history | grep docker

Tìm lệnh trong history.

Lỗi thường gặp

Lỗi Nguyên nhân Cách sửa
command not found chưa cài package cài package
permission denied thiếu quyền dùng sudo
no such file sai path kiểm tra ls
ký tự lạ encoding sai kiểm tra locale
