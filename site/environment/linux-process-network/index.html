Process Và Network Cơ Bản

Phần này tập trung vào process, port và các lệnh network thiết yếu.

4. Quản lý process

Xem process

ps aux

Tìm process

ps aux | grep python

Xem realtime

htop

hoặc

top

Kill process

kill 12345

Force kill:

kill -9 12345

Kill theo tên

pkill -f "node server.js"

Chạy process background

python app.py &

Đưa process về foreground

fg

5. Network cơ bản

Xem IP

Linux:

ip addr show

macOS:

ifconfig

Kiểm tra kết nối

ping google.com -c 4

Xem port đang mở

Linux:

ss -tulnp

macOS:

lsof -i :8080

Kiểm tra service

curl -I http://localhost:3000

DNS lookup

nslookup example.com

hoặc

dig example.com

Download file

wget https://example.com/file.zip

hoặc

curl -O https://example.com/file.zip
