Quản lý File Trong Terminal

Phần này gom các thao tác làm việc với file, thư mục và đọc nội dung file.

Quản lý file

Tạo thư mục

mkdir project

Tạo nhiều cấp:

mkdir -p project/src/utils

Tạo file

touch index.js

Copy file

cp file.txt backup.txt

Copy folder:

cp -r src/ src_backup/

Di chuyển / đổi tên

mv old.txt new.txt

Xoá file

rm file.txt

Xoá folder

rm -rf folder/

Cảnh báo

Luôn kiểm tra kỹ path trước khi dùng rm -rf.

Xem nội dung file

In toàn bộ file

cat file.txt

Xem 20 dòng đầu

head -20 file.txt

Xem 20 dòng cuối

tail -20 file.txt

Theo dõi log realtime

tail -f app.log

Xem file theo trang

less file.txt

Thoát bằng:

q

Đếm số dòng

wc -l file.txt
