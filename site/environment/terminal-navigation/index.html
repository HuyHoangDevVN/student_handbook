Điều hướng Terminal

Phần này tập trung vào các thao tác điều hướng thư mục cơ bản trong terminal.

Điều hướng thư mục

In thư mục hiện tại

pwd

Liệt kê file

ls

Xem chi tiết:

ls -la

Di chuyển thư mục

cd folder

Lên một cấp

cd ..

Về home directory

cd ~

Quay lại thư mục trước

cd -
