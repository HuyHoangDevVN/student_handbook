# Docker co ban

Trang nay gio la muc luc cho cum bai Docker. Noi dung duoc tach thanh cac bai nho de nguoi hoc di
tung buoc tu runtime -> debug/storage -> Dockerfile/build.

---

## Muc tieu

Sau cum bai nay, ban co the:

- hieu image, container, volume va network
- chay va quan ly container bang `docker run`
- doc logs, debug container va lam viec voi volume
- build image bang Dockerfile
