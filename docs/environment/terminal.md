# Terminal & Shell co ban

Trang nay gio dong vai tro muc luc cho cum bai ve terminal. Noi dung chi tiet da duoc tach nho
de hoc theo tung ky nang thay vi doc mot bai qua dai.

---

## Muc tieu

Sau cum bai nay, ban co the:

- phan biet terminal va shell
- di chuyen thu muc va lam viec voi file bang command line
- ket hop lenh bang `pipe` va `redirect`
- tim file, tim text va quan ly bien moi truong
