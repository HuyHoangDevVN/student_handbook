# Điều hướng Terminal

Phần này tập trung vào các thao tác điều hướng thư mục cơ bản trong terminal.

---

## Điều hướng thư mục

### In thư mục hiện tại

```bash
pwd
```

---

### Liệt kê file

```bash
ls
```

Xem chi tiết:

```bash
ls -la
```

---

### Di chuyển thư mục

```bash
cd folder
```

---

### Lên một cấp

```bash
cd ..
```

---

### Về home directory

```bash
cd ~
```

---

### Quay lại thư mục trước

```bash
cd -
```

---
